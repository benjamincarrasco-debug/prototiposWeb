Introducción al lenguaje

HTML es el lenguaje de marcado estándar utilizado para crear y estructurar el contenido de las páginas web. Sus siglas significan HyperText Markup Language (Lenguaje de Marcado de Hipertexto).
A diferencia de los lenguajes de programación que ejecutan funciones lógicas, el HTML se encarga de definir el "esqueleto" de un sitio. Funciona mediante un sistema de etiquetas (tags) que indican al navegador cómo debe mostrar elementos como textos, imágenes o enlaces.

Etiquetas de Formato

Función visual: Por defecto, los navegadores la renderizan como una línea horizontal que cruza el ancho del contenedor.
Significado semántico: Representa un cambio de tema o una transición en la narrativa del documento.
